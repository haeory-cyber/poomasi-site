# 사교원 사이트 관리 안내서

> 이 문서를 Claude에게 붙여넣으면 사교원 사이트 수정을 요청할 수 있습니다.

---

## 사이트 개요

사회혁신교육원 사회적협동조합(사교원) 홈페이지입니다.

- 메인: https://sakyowon.poomasi.org/
- 햇빛소득마을 업무도우미: https://sakyowon.poomasi.org/sunvillage.html

## 파일 구성

사이트는 3개 파일로 되어있습니다.

- **index.html** — 메인 홈페이지 (소개, 사업, 프로젝트, 조직, 문의 폼)
- **sunvillage.html** — 햇빛소득마을 종합업무도우미 (9개 탭, AI 기능 포함)
- **server.py** — 웹서버 (Flask, 문의 저장 + AI 프록시)

CSS와 JS는 각 HTML 파일 안에 내장되어 있습니다. 별도 파일 없습니다.

## 수정 가능한 것

- 텍스트 내용 (법인 소개, 사업 설명, 프로젝트 현황 등)
- 조직 구성원 정보
- 통계 숫자 (매출, 인원 등)
- 디자인/스타일 (CSS)
- 문의 유형 추가
- 새 페이지 추가 (HTML 파일)

## 수정하면 안 되는 것

- API 키, 비밀번호, .env 파일 관련 내용
- index.html 맨 아래 `<script src="https://seed.poomasi.org/poomai-widget.js"></script>` 줄 (삭제/수정 금지)
- 서버 인프라 (도메인, DNS, SSL, 터널 설정)

이런 건 본부(지미)한테 요청해야 합니다.

## 콘텐츠 위치 안내

index.html 안에서 수정할 위치를 찾을 때:

| 수정 내용 | 찾을 키워드 |
|----------|-----------|
| 법인 소개 | `id="about"` |
| 주요 사업 3개 카드 | `id="business"` |
| 세부 프로젝트 목록 | `id="projects"` |
| 조직 구성원 | `id="org"` |
| 문의 폼 유형 | `id="inq-category"` |
| 통계 (매출, 인원 등) | `class="about-stats"` |
| 슬로건 | `class="hero-slogan"` |

## 수정을 요청하는 법

이 문서 전체를 Claude에 붙여넣은 뒤, 원하는 수정사항을 자연어로 말하면 됩니다.

예시:
- "프로젝트 현황에 '순천 교육사업' 항목 추가해줘. 기간은 2026.03~2026.08, 예산 5천만원, 진행중"
- "이사회 명단에서 김영구 이사를 빼줘"
- "매출 숫자를 15억으로 바꿔줘"
- "햇빛소득마을 페이지에서 대시보드 탭 제목을 '현황판'으로 바꿔줘"

Claude가 수정된 HTML 코드를 만들어주면, 그 코드를 본부(지미)에게 전달하거나 직접 파일에 반영하면 됩니다.

## 서버 관련 (참고)

- 서버는 GCP에서 자동 운영됩니다
- HTML 파일을 수정하면 바로 반영됩니다 (서버 재시작 불필요)
- server.py를 수정한 경우에만 재시작 필요: `sudo systemctl restart sakyowon-web.service`
- 문의 폼 데이터는 Supabase에 저장됩니다
- 서버 문제는 본부(지미)에게 연락하세요
